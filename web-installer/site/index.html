CYD-S3 Stereo

Firmware installer for the Freenove FNK0104S internet radio player. Bundled firmware version: …

Install over USB

  1. Connect the board with a data-capable USB-C cable.
  2. Click Connect below and pick the serial port (USB JTAG/serial debug unit or similar).
  3. Choose Install. A full install takes about a minute.
This browser lacks Web Serial — use Chrome or Edge on desktop.
Web Serial needs a secure context. Open this page via http://localhost:… on the machine the board is plugged into, or serve it over HTTPS.

Board not detected? Hold BOOT, tap RESET, release BOOT, then retry. An install erases the device — a previously configured player will need setting up again.

After flashing

  1. The device boots into setup mode and starts a WiFi access point named CYD-Radio-xxxx.
  2. Join it from a phone or laptop; the setup page opens automatically (or browse to 192.168.4.1).
  3. Enter WiFi credentials, stream URLs, and the update-server address — future updates then arrive over the air.
