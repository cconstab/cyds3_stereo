Flash a new board

Flashes the current published firmware to a factory-fresh Freenove FNK0104S over USB. Requires Chrome or Edge (Web Serial).

  1. Connect the board with a data-capable USB-C cable.
  2. If the board is brand new it should just work; if the connection fails, hold BOOT, tap RESET, release BOOT, then retry.
  3. Click Connect, pick the serial port, and install.

After flashing, the device starts a WiFi access point named CYD-Radio-xxxx. Join it from a phone and the setup page opens (or browse to 192.168.4.1) to configure WiFi and stream URLs.
